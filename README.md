# Music Video Jukebox

A music video jukebox application that allows users to search for and play music videos from various artists.

## Features

- Search for artists and their music videos
- Play videos directly from YouTube
- Clean and modern user interface
- Auto-play functionality
- Queue management

## Setup

1. Clone the repository:
```bash
git clone https://github.com/yourusername/music-video-jukebox.git
cd music-video-jukebox
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```

## Data Structure

The application uses a JSON file (`Videos_by_Artist.JSON`) as its database, which contains:
- Artist information
- Music video URLs
- Thumbnail URLs
- Official Artist Channel information (when available)
